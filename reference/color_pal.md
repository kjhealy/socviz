# Draw a palette of colors

Draw a palette of colors

## Usage

``` r
color_pal(col, border = "gray70", ...)
```

## Arguments

- col:

  vector of colors

- border:

  border

- ...:

  other arguments

## Value

Plot of a color palette

## Details

Borrowed from the colorspace library

## Author

colorspace library authors

## Examples

``` r
color_pal(c("#66C2A5", "#FC8D62", "#8DA0CB"))
```
