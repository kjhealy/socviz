# Student debt data

Outstanding student debts in 2016 across 8 income categories, by percent
of all borrowers and percent of all balances.

## Usage

``` r
studebt
```

## Format

\## \`studebt\` A data frame with 16 rows and 4 columns:

- Debt:

  Debt categories (character)

- type:

  Pct in terms of Borrowers or Balances

- pct:

  Percentage of all type

- Debtrc:

  Debt categories (ordered factor)

## Source

Federal Reserve Bank of New York.
